Principais Formatações:

Negrito/Destaque

Para deixar um texto em negrito utiliza-se a tag <b></b>.

A tag <strong></strong> também pode ser utilizada para a mesma função.

Itálico/Ênfase

As tags <i></i> e <em></em> são utilizadas para deixar um texto em itálico.

Texto Marcado

"A medida do amor é amar sem medida."

Ao marcar um trecho de um texto utiliza-se a tag <mark></mark>.

Texto Pequeno

Pode-se deixar o texto com a letra pequena utilizando a tag <small></small>.

Texto Excluído

Desejando excluir algumas palavras utiliza-se a tag <del></del>.

Texto Sublinhado

Utiliza-se a tag <ins></ins> para sublinhar alguma palavra.

Texto Sobrescrito

O texto sobrescrito é inserido através da tag <sup></sup>, como é apresentado em fórmulas matemáticas. Ex.: x5.

Texto Subescrito

O texto sobrescrito é inserido através da tag <sub></sub>, como é apresentado em fórmulas químicas. Ex.: H2O.

Citação Simples

A tag <q></q> antecipa uma citação. Como dizia o filósofo: O sábio sabe tudo o que diz, mas não diz tudo o que sabe.

Abreviações

Para indicar no documento HTML uma abreviação utiliza-se a tag <abbr></abbr>.
